# Helldiver Database

## Project Overview

The Helldiver Database is a full-stack web application designed to provide structured, searchable, and interactive access to game data from Helldivers 2.

The system will function as a centralized database for:

- Weapons (damage, fire rate, armor penetration, type)
- Stratagems (cooldown, category, uses)
- Enemies (faction, armor level, weak points)
- Weapon–enemy effectiveness relationships

Unlike traditional wiki pages, this project focuses on interactive filtering, relational data modeling, and comparative analysis tools.

## Planned Features

- Search and filter weapons by stats
- View enemies categorized by faction
- Browse stratagems with cooldown data
- Compare multiple weapons side-by-side
- View recommended counters for enemy types
